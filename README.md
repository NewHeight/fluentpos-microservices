- RMQ for Queues / Messaging

- CI/CD on Github Actions
- Terraform Support
- Kubernetes
- Postgre & MongoDB Datastore
- Redis Cache
- ELK Stack
- Push Docker images to public Docker Hub Repos
- Postman Collection for Testing
- VS Code as Primary IDE
